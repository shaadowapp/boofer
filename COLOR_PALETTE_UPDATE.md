# Color Palette - Intimate Couples Messaging App

## App Vision
A private messaging app for couples combining WhatsApp's functionality with Snapchat's intimacy and top-tier privacy features. The color palette reflects romance, trust, and security.

## Design Philosophy

### Core Values Through Color
- **Romance & Intimacy**: Warm pinks and roses create emotional connection
- **Privacy & Security**: Blues and greens signal trust and encryption
- **Modern & Playful**: Snapchat-inspired vibrancy with WhatsApp familiarity
- **Couple-Focused**: Colors that feel personal and intimate, not corporate

### Color Psychology
- **Love Rose (#FF6B9D)**: Main brand - warmth, love, connection
- **Deep Blush (#E91E63)**: Passion, highlights, important actions
- **Trust Blue (#2196F3)**: Security, privacy, reliability
- **Secret Green (#00C853)**: Encryption, secure status, success

## Color Palette

### Brand Colors - Romance & Security
- **Love Rose**: `#FF6B9D` - Primary brand color for romantic elements
- **Deep Blush**: `#E91E63` - Accents, highlights, passionate moments
- **Trust Blue**: `#2196F3` - Privacy indicators, security features
- **Secret Green**: `#00C853` - Encrypted status, secure connections

### Light Mode - "Romantic Warmth"
- **Background**: `#FFF8F8` - Soft rose tint for warmth and comfort
- **App Bar**: `#FFFFFF` - Clean white for clarity
- **Search Bar**: `#F5F5F5` - Subtle grey for gentle contrast
- **Your Messages**: `#FFE0E6` - Light rose bubble (sender)
- **Partner's Messages**: `#F0F0F0` - Soft grey bubble (receiver)
- **Primary Text**: `#2C2C2C` - Warm dark grey for readability
- **Secondary Text**: `#757575` - Medium grey for timestamps

### Dark Mode - "Intimate Night"
- **Background**: `#121212` - True dark for privacy and intimacy
- **App Bar**: `#1E1E1E` - Elevated surface for depth
- **Search Bar**: `#2C2C2C` - Subtle contrast for search
- **Your Messages**: `#4A1942` - Deep rose bubble (sender)
- **Partner's Messages**: `#2C2C2C` - Dark grey bubble (receiver)
- **Primary Text**: `#FFFFFF` - Pure white for clarity
- **Secondary Text**: `#B0B0B0` - Light grey for timestamps

### Feature-Specific Colors
- **Online Status**: `#00C853` - Secret Green for active/online indicators
- **Encrypted Messages**: `#2196F3` - Trust Blue for end-to-end encryption
- **Heart Reactions**: `#FF6B9D` - Love Rose for likes and hearts
- **Disappearing Messages**: `#FF9800` - Orange for temporary content
- **Private Mode**: `#6A1B9A` - Deep purple for enhanced privacy
- **Biometric Lock**: `#4CAF50` - Green for fingerprint/face unlock

## Key Features

### Romantic Elements
1. **Warm Backgrounds**: Rose-tinted light mode feels intimate and welcoming
2. **Heart-Themed FAB**: Love Rose floating action button for new messages
3. **Couple Bubbles**: Distinct colors for you vs your partner's messages
4. **Romantic Accents**: Pink highlights for special moments and reactions

### Privacy & Security
1. **Trust Indicators**: Blue colors signal encrypted and secure features
2. **Dark Intimacy**: True dark mode for private late-night conversations
3. **Security Status**: Clear visual indicators for encryption and privacy modes
4. **Biometric Integration**: Green accents for secure authentication

### Modern Messaging
1. **WhatsApp Familiarity**: Clean, readable message bubbles and layout
2. **Snapchat Playfulness**: Vibrant colors for reactions and special features
3. **Status Clarity**: Distinct colors for online, encrypted, and disappearing content
4. **Accessibility**: High contrast ratios and clear visual hierarchy

## Technical Implementation

### Context-Aware Functions
- `scaffoldBackground(isDark)` - Romantic warmth vs intimate night
- `headerBackground(isDark)` - Clean white vs elevated dark
- `senderBubble(isDark)` - Light rose vs deep rose for your messages
- `receiverBubble(isDark)` - Soft grey vs dark grey for partner's messages
- `floatingActionButton(isDark)` - Love Rose vs Deep Blush heart button

### Feature-Specific Colors
- `onlineStatus` - Secret Green for active indicators
- `encryptedStatus` - Trust Blue for security features
- `heartReaction` - Love Rose for romantic interactions
- `disappearingMessage` - Orange for temporary content
- `privateMode` - Deep purple for enhanced privacy

### Color Extensions
- `withRomanticAccent` - Adds Love Rose tint to elements
- `withTrustAccent` - Adds Trust Blue for security features
- `withWarmGlow` - Creates intimate glow effects

## Usage Guidelines

### Romantic Features:
- Use Love Rose for hearts, likes, and romantic interactions
- Deep Blush for passionate moments and important actions
- Warm backgrounds create intimate, couple-focused atmosphere

### Privacy Features:
- Trust Blue for all security and encryption indicators
- Secret Green for secure connections and verified status
- Deep purple for private mode and enhanced security

### Message Design:
- Your messages: Light rose (light) / Deep rose (dark)
- Partner's messages: Soft grey (light) / Dark grey (dark)
- Clear distinction without overwhelming brand presence

The palette creates an intimate, secure messaging experience that feels designed specifically for couples while maintaining the familiarity of popular messaging apps!
